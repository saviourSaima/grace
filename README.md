# React Dashboard Application

A production-ready React dashboard application with form controls, dropdowns, and accordion components.

## Features

- Search dropdown in form
- Accordion sections with nested dropdowns
- Bootstrap styling for responsive design
- Modern React practices

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Technologies

- React 17+
- React Bootstrap
- Bootstrap 5# grace
